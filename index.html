What's New

Changelog

Every update to Don't Order Wrong, newest first.

v0.9 LATEST March 2025
  • Added Edit Entry — update notes, scores, photos and more on any existing entry
  • Unique row IDs added for reliable edit/update targeting in the database
  • Custom tags — type and add your own tags beyond the preset list
  • Removed Noise Level from aspect scoring (now: Cuisine, Atmosphere, Service, Value)
  • Date and Cuisine fields fixed to sit side-by-side cleanly on mobile
  • Download the App button — install DOW as a home screen icon on any device
  • Changelog page added (you're looking at it!)
  • Swipe right from left edge to go back on any screen
  • Suggestion Box added to profile — email the developer directly
v0.8 February 2025
  • Username accounts — each person gets their own private journal
  • Sign up and sign in with just your name, no password needed
  • Switch Journal — visit another user's journal by entering their name
  • Entries now tagged with username so journals stay separate
v0.7 February 2025
  • PWA support — app can be installed on iPhone and Android home screens
  • DOW icon designed and added across all icon sizes
  • Service worker added for offline support and instant loading
v0.6 February 2025
  • Photo uploads via Cloudinary — photos now sync across all devices
  • Photo gallery in entry detail view with lightbox tap-to-expand
  • Custom domain live at dontorderwrong.com
v0.5 February 2025
  • App renamed to Don't Order Wrong dot com
  • Bebas Neue logo typeface added
  • Google Maps integration — each entry links to restaurant location
  • Profile tab with dining stats (visits, average score, top cuisine)
  • Bottom navigation bar added
v0.1 – v0.4 February 2025
  • Initial concept and prototype built
  • Google Sheets + SheetDB backend integration
  • 10-point rating system with aspect breakdowns
  • Tag system, companion tracking, notes field
  • Deployed to GitHub Pages
